# Presentation

Материалы для 5-минутного выступления по курсовой работе.
Структура презентации выстроена по критериям методических указаний: данные, подготовка, split и метрики, сравнение подходов, несколько моделей, подбор гиперпараметров, контроль переобучения и выводы.

- `slides.md` — исходник презентации.
- `presentation.pptx` — готовая презентация на 12 слайдов с титульным.
- `speech.md` — текст для произношения.
- `assets/` — PNG-диаграммы для презентации и markdown-просмотра.
- `assets/notebook/` — графики, извлеченные из output-ячееек Jupyter Notebook без перерисовки.

Сборка презентации:

```bash
pandoc slides.md -t pptx --slide-level=1 -o presentation.pptx
```
