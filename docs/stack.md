# SciGuide — стек проекта

## Backend

- Python
- FastAPI
- Чистая архитектура
- Разделение на доменные области

## Frontend

- Vue
- FSD

## Данные и хранилища

- PostgreSQL — основное хранилище
- Redis — кэш и брокер фоновых задач
- Qdrant — семантический поиск
- Neo4j — графовая модель
- MinIO — файловое хранилище

## Миграции

- Flyway

## Пайплайны и обработка

- LangChain Text Splitters — нарезка текста
- NetworkX — работа с графовыми структурами
- Celery — фоновые задачи

## Инфраструктура

- Docker — контейнеризация
- Docker Compose — оркестрация
- Nginx — reverse proxy
- GitHub Actions — CI/CD

## Observability

- Prometheus
- Grafana
- Loki
- Grafana Alloy
- OpenTelemetry — формат логов и телеметрии
